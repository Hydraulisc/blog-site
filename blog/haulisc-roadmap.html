haulisc-roadmap: A Vague Hydraulisc Roadmap

Welcome to another Developers Digest.

This entry provides a vague rundown of the Hydraulisc Foss Rewrite Roadmap.

Initially, we aim for a 12-18 month turnover.

January 1st 2025

Pre-Build Phase

Local development. Invite and Role features implementations.

January 8th 2025

Initial Commit

First official commit to the new FOSS Repository on GitHub.

January 9th 2025

Initial APT Rollout

First official APT commit, after adding isAdmin features.

January 12th 2025

APT Phase

Alpha Private Testing Phase. Database will be deleted multiple times. Bug Hunting with a select team of Alpha Testers. Renewed welcome page.

First official, private and invite-only, release.

January 19th - February 10th 2025

Renew Settings

Renewed settings page, based off of old one.

Prefers HTML form over JavaScript FETCH.

January 26th 2025

File Upload Functionality

Allowed Post uploads and avatar/banner uploads.

Also checks file mimetypes, and sanitizes HTML before storing or rendering.

February 3rd 2025

PAT/PBT Phase

Public Alpha Triage Phase transitioning to Public Beta Testing phase.

Larger group of Alpha testers and triage of Beta testers as we transition to a Public Beta.

February 9th 2025

PBT Phase

Fully Public Beta Test, as we transition to a full release.

Removing invite-only mode.

April 2025

OAuth2 Handles

Allow users to register for an OAuth2 client on Hydraulisc from a "developers" section in settings.

OAuth2 scopes restricted to "identify" and "connections" for third parties.

February 2025 - October 2025

Theme Implementations

Ability to switch between themes.

Adding custom CSS support. Major CSS overhaul.

N/A

Language Implementations

Community Translations!

Pending 3rd party FOSS software release!

N/A

Final Product Release

Full release of the FOSS Hydraulisc Rewrite on GitHub.

This is Not Applicable (N/A) because Hydraulisc will be continuously developed & upgraded.
