Dependency Checks

Welcome to another Developers Digest.

Today's topic: Dependencies.

As many of you should know, projects like Hydraulisc require dependencies to run smoothly.

Recently, we had to do a lot of security patches; mainly due to CVE-2025-48387, which affected tar-fs.

Also, appreciating the opportunity to mention subtle changes, we've updated Blogger, cleaned the code a bit, and updated to a more recent version of the Firebase JS SDK.

We've also recently introduced a new member to the Dev Team; Arkarivs, a Spanish developer who's helping make StarPlace.

Important update (2025-10-21):

As of this commit, Blogger has been updated even further, due to Google limiting Firestore databases to 1MB and removing all free storage from Firebase.

This new update to Blogger allowed us to make the migration from a Firebase Firestore to a GitHub-Native blog.
